# AI产量预测智能体 - 快速演示指南 🚀

## 📱 如何打开原型

### 方法一:双击打开(推荐)
1. 双击 `index.html` 文件
2. 系统会自动用默认浏览器打开

### 方法二:使用批处理文件
1. 双击 `open.bat` 文件
2. 自动在浏览器中打开

### 方法三:使用本地服务器(最佳体验)
1. 双击 `start.bat` 文件
2. 选择启动方式(推荐Python)
3. 在浏览器中访问 `http://localhost:8000`

## 🎯 5分钟快速演示流程

### 演示路线一:播种前决策(3分钟)

#### 第1步:进入智能体(30秒)
```
从左侧导航 → AI → 产量预测智能体
```
- **看点**: 智能体介绍卡片(紫色渐变,大脑图标)
- **说明**: "基于大数据分析,为您提供作物产量与产值预估"

#### 第2步:选择场景(10秒)
```
点击"播种前·辅助决策"卡片
```
- **看点**: 两个功能卡片,橙色和紫色渐变区分
- **说明**: "想知道种什么能赚钱?"

#### 第3步:填写信息(1分钟)
```
作物类型: 选择"小麦"
种植区域: 河南省 → 商丘市 → 柘城县
播种时间: 选择"10月(秋季)"
```
- **看点**: 三级联动选择(选择省份后城市自动更新)
- **说明**: "智能三级联动,操作便捷"

#### 第4步:查看分析(1分钟)
```
点击"开始分析"按钮
```
- **看点**: 
  - 加载动画(旋转图标+3个步骤)
  - 结果页面展示:
    - 最低产量: 700斤/亩(橙色)
    - 最高产量: 950斤/亩(绿色)
    - 平均产量: 825斤/亩(紫色标注)
- **说明**: "基于柘城县近5年小麦种植历史数据"

#### 第5步:后续操作(30秒)
```
可选择"预估地块产值"继续演示
```

### 演示路线二:地块产值预估(5分钟)⭐推荐

#### 第1步:进入智能体(30秒)
```
从左侧导航 → AI → 产量预测智能体
```

#### 第2步:选择场景(10秒)
```
点击"种植中·地块产值预估"卡片
```
- **看点**: 紫色渐变图标,更贴近实际应用
- **说明**: "我的地能卖多少钱?"

#### 第3步:选择地块(30秒)
```
选择第一个地块:"大厅水培植物 - 一号分区"
```
- **看点**: 
  - 地块卡片展示完整信息
  - 位置、面积、作物、生长状态
  - 悬停效果(边框变紫色)
- **说明**: "系统自动关联地块信息"

#### 第4步:查看地块详情(1分钟)
```
地块详情页展示
```
- **看点**: 
  - 6个信息项(位置/面积/作物/播种日期/生长阶段/预计收获)
  - 补充信息选项卡(语音/拍照)
- **说明**: "可选择补充更多信息提高准确度"

#### 第5步:补充信息(可选,30秒)
```
点击"语音描述" → 输入:"长势良好,每亩约4000株"
或
点击"拍照上传" → 选择图片上传
```
- **看点**: 
  - 展开/收起动画
  - 图片预览功能
- **说明**: "支持多种方式补充信息"

#### 第6步:生成报告(2分钟)⭐核心展示
```
点击"开始预测产值"
```

**加载动画(2秒)**
- 显示计算步骤

**智能报告展示(重点!)**

1. **核心数据卡片(最突出)**
   - 预计总产值: ¥4,505元(大字号,紫色渐变背景)
   - 4个小数据:
     - 预计亩产: 850斤
     - 预计总产: 1,700斤
     - 当前市价: ¥2.65/斤
     - 亩均产值: ¥2,252

2. **地块信息**
   - 完整展示地块基本信息

3. **气象分析**
   - 3个分析项(带图标和颜色标识)
   - ✅ 近期气温适宜,有利于作物生长(绿色)
   - ✅ 降水充足,土壤墒情良好(绿色)
   - ℹ️ 未来15天无极端天气预警(蓝色)

4. **价格趋势**
   - 当前价格: ¥2.65/斤
   - 近期走势: 稳中有升(绿色上箭头)
   - 市场预期: 收获期价格稳定

5. **管理建议**
   - 4条具体建议(带勾号)
   - 田间管理/气象预警/病虫害防治/销售时机

6. **免责声明**
   - 黄色提示框
   - "本报告基于模型估算,数据仅供参考"

#### 第7步:后续操作(30秒)
```
可选择:
- "返回地块列表" → 查看其他地块

```

## 🛠️ 常见演示问题处理

### Q:页面样式不对?
A:刷新页面(F5),确保CSS文件已加载

### Q:选择省份后没有城市?
A:这是正常的,先选择河南省,然后会出现城市列表

### Q:照片上传后看不到?
A:照片会在预览区显示,这是前端演示,不会真的上传

### Q:分享按钮点击无反应?
A:这是预留功能,会提示"开发中"

### Q:数据从哪里来?
A:这是原型演示,使用的是模拟数据
