1. 访问你的 GitHub 仓库：
   ```
   https://github.com/DPXJ/TYKJ-AI-chan_liang
   ```

2. 进入 **Settings** → **Pages**

3. 确认配置如下：
   - **Source**: Deploy from a branch
   - **Branch**: main
   - **Folder**: / (root)

4. 保存后，GitHub 会自动部署

## 📞 常见问题

### Q1: 推送后页面没有更新？

**A**: 
1. 等待2-3分钟让 GitHub Actions 完成构建
2. 按 Ctrl+Shift+R 强制刷新浏览器
3. 检查 GitHub 仓库的 Actions 标签，确认部署成功

### Q2: 如何查看部署状态？

**A**: 访问 `https://github.com/DPXJ/TYKJ-AI-chan_liang/actions`
- 绿色 ✓ = 部署成功
- 黄色 ⏳ = 正在部署
- 红色 ✗ = 部署失败

### Q3: 访问页面出现404？

**A**: 
1. 确认 GitHub Pages 已启用（Settings → Pages）
2. 确认分支是 main 而不是 master
3. 等待首次部署完成（约5分钟）
